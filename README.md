# 使用说明
下载后直接点击静态粘贴.exe就能使用。 打开后主界面会在屏幕的正上方

第一个圆为反选按钮
第一个方形按钮为清空勾选项记录按钮。
  勾选对应的数据后点击方形按钮可以删除
 
双击软件上的F1-F6可以锁定/解锁文本，文本锁定后文本变成红色且不会被快捷键清除，但可以勾选后点方形按钮清除

快捷键ALT+D：可以清除所有没有被锁定的文本，即黑色的文本
快捷键ALT+Q：显示/隐藏本软件
快捷键F1-F6：粘贴对应的文本

如果提示快捷键注册失败的，可以解决冲突后，重新打开本软件就会自动注册快捷键了。
